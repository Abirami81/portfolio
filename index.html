Intellipark Mapview

Intellipark Mapview combines grid-based pathfinding with cloud integration. It automatically assigns drivers to available parking slots, computes the shortest path using the A* algorithm, and visually highlights the route in green on a parking layout. The generated layout is uploaded to the cloud, and a QR code is produced so drivers can scan it and view the personalized parking map on their phones. This ensures efficient slot allocation, easy navigation, and a seamless user experience.


Key Features:
• Grid-Based Parking Layout
• A* Pathfinding Algorithm
• Sequential Slot Assignment
• Visual Path Highlighting
• Cloud Integration (ImgBB/Catbox)
• QR Code Generation

Water Quality Monitoring System

This project introduces a Machine Learning–based water quality monitoring system that evaluates purity using parameters like pH, TDS, turbidity, and temperature. By applying Bayesian Change Point Detection (BCPD) on a large dataset, the system achieves near-perfect accuracy in predicting whether water samples meet WHO safety standards. It also integrates real-time alerts via Gmail and voice notifications, offering a cost-effective and automated solution for continuous water quality monitoring.


Key Features:
• Automated Monitoring
• BCPD Model
• Multi-parameter Analysis
• Performance Evaluation
• WHO Standards Compliance
• Smart Alerts

Caner Cell Detection

This project presents an intelligent cancer cell detection system that leverages Convolutional Neural Networks (CNN) and traditional Machine Learning algorithms to accurately differentiate between cancerous and non-cancerous cells using microscopic image analysis.The model analyzes histopathological images and identifies abnormal cell structures based on color variations, texture patterns, and morphological features. By utilizing color differentiation techniques, the system enhances classification accuracy and supports early-stage cancer diagnosis.


Key Features:
• CNN-Based Image Classification
• Color Differentiation Analysis
• Machine Learning Integration
• Feature Extraction
• Performance Evaluation Metrics
• Automated Detection System
